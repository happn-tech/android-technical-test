
# Happn Android Technical Test Back-End

Back-End for Happn Android Test

### Prerequisites

You need for installing this back-end :

- NodeJS 

- MongoDB server

### Installing
  
- Start MongoDB server 
  
```
mongod
```

- Install npm dependencies
```
npm install
```  

- And then, start the server
  
```
npm start
```
### Test
```
npm test
```
